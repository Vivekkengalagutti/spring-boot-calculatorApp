# Spring Boot Calculator

A simple calculator web application built using Spring Boot and Thymeleaf.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling (division by zero)

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- HTML/CSS

## How to Run
1. Clone the repository
2. Run the application
3. Open http://localhost:8080
